# Platform UI kit — sample screen

The source spec (`uploads/design-system.md`) deliberately defines **no page layouts** — "page structure comes with each request". This kit therefore contains a single illustrative composition, not a recreation: a profile-form step + document dossier built ONLY from spec-defined chrome and components (TopBar, Card, Field, Button, StatusBadge/Pill, Flag, Icon) on the `--paper` background.

Use it as a reference for how the pieces compose; style any requested page from the tokens and components, per the spec's checklist (§15).

- there is no search bar in the topbar in the platform.
